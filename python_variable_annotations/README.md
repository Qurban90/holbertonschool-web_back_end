# Python - Variable Annotations

This project introduces type annotations in Python 3. Type annotations allow developers to specify the expected types of function parameters and return values, improving code readability and enabling static type checking with tools like `mypy`.

## Learning Objectives

- Type annotations in Python 3
- How to use type annotations to specify function signatures and variable types
- Duck typing
- How to validate your code with `mypy`

## Requirements

- All files are interpreted/compiled on Ubuntu 18.04 LTS using Python 3.7
- All files end with a new line
- The first line of all files is exactly `#!/usr/bin/env python3`
- All files must be executable
- Code follows the `pycodestyle` style (version 2.5)
- All modules, classes, and functions have documentation
- All functions and coroutines are type-annotated
